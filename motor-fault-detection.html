Motor Fault Detection via Vibration Analysis

A comprehensive machine learning system utilizing vibration analysis to identify motor faults including misalignment, unbalance, and bearing defects for predictive maintenance in industrial applications. This project demonstrates the integration of signal processing, feature extraction, and multiple ML algorithms for reliable fault classification.

Project Overview

This project focuses on Predictive Maintenance, used to identify issues like misalignment, unbalance, and bearing defects early to prevent unexpected machine failures and costly downtime. The key problem addressed is the timely maintenance of electric motors can be vital for complex industrial production processes. Unexpected motor failures cause production downtime and costly repairs resulting in lower efficiencies and losses for the organization in general.

Relevance to Mechatronics

Integrates actuators, signal processing and machine learning to build a real-time, Motor Vibration analysis based on axis acceleration data and demonstrating a complete Mechatronics application.

Dataset & Methodology

Dataset Composition

The dataset used in this project consists of 4,000 samples and 4 fault types: Normal (1,000 samples), Misalignment (1,000 samples), Unbalance (1,000 samples), and Bearing fault (1,000 samples).

Training Configuration

For our project, from feature extraction all 4000 samples were used, but for our model training purposes using the algorithms we used to conclude our accuracy, 800 samples from each class were loaded which contains X,Y,Z axis data within it, resulting in 2400 samples for each class.

Feature Extraction

Comprehensive feature extraction combining time-domain, frequency-domain, and shape factors to create a robust representation of vibration signatures for accurate fault classification.

Signal Processing

FFT processing to remove high frequency data, normalization techniques, and PCA analysis for optimal feature selection and dimensionality reduction.

My Primary Contribution - K-Nearest Neighbors (KNN) Implementation

KNN Algorithm Development & Optimization

I was primarily responsible for implementing and optimizing the K-Nearest Neighbors algorithm for this motor fault detection system. My work involved comprehensive data preprocessing, hyperparameter tuning, and performance evaluation.

Data Preprocessing Pipeline
  • Drop NaN values
  • Check for duplicate samples
  • Normalize per sample
  • FFT processing (remove high frequency data)
  • Extract features (time domain, frequency domain, shape factors)
  • PCA analysis
  • Label encoding of target
  • 80-20 Train-test split
Hyperparameter Optimization
  • k values from 1 to 30
  • Final optimal k = 3
  • metric = Manhattan
  • weights = uniform
  • algorithm = auto

Trained model with multiple k values and evaluated using accuracy, confusion matrix, and classification report.

Web Application Deployment

To demonstrate the practical applicability of the developed XGBoost model, a comprehensive web-based user interface was implemented using Flask framework. The web application follows a client-server architecture with the following components:

Backend Implementation

Flask web framework for HTTP request handling and API endpoints, Joblib library for model serialization and loading, NumPy and Pandas for data preprocessing and numerical computations, JSON-based communication protocol for data exchange.

Frontend Interface

Responsive HTML5 interface with CSS styling for professional appearance, JavaScript-based form handling and asynchronous API communication, Real-time prediction display with confidence scores and probability distributions.

User Input Features

The web interface provides an intuitive input form where users can enter the six critical features (RMS, Mean Frequency, Dominant Frequency, Crest Factor, Shape Factor, Axis Encoding) identified during model development.

Prediction Results

Primary Classification: Clear identification of the detected fault type (Normal Operation, Misalignment, Unbalance, or Bearing fault), Confidence Level, Probability Distribution, and Timestamp recording.

Key Findings & Technical Achievements

This research successfully demonstrates that machine learning can be effectively applied to motor fault detection using vibration analysis. The evaluation of four different algorithms provides practical guidance for implementing predictive maintenance systems in industrial settings.

  • KNN Performance: Achieved 93% accuracy with k=3. The Manhattan distance metric worked well for our vibration feature space, and the fast training time is advantageous. However, the lower accuracy suggests that the assumption of local similarity in the feature space may not fully capture the complex patterns present in motor fault data.
  • Feature Engineering Excellence: Our approach of combining time-domain, frequency-domain, and shape factors created a comprehensive representation of the vibration signatures. The EDA confirmed that dominant frequency was the most informative feature, which aligns with established principles in vibration analysis.
  • Industrial Application: The accuracy levels achieved by three of the four models (exceeding 93%) demonstrate that machine learning-based fault detection is ready for practical implementation. Such systems could significantly reduce unexpected motor failures and the associated costs of unplanned downtime in industrial facilities.
  • Mechatronics Integration: This project represents a successful integration of multiple mechatronics disciplines - combining sensor technology for data acquisition, signal processing techniques for feature extraction, and machine learning for intelligent decision-making.

The resulting system embodies the principles of Industry 4.0 and demonstrates how predictive maintenance can be practically implemented using modern data analysis techniques. The comprehensive comparison of algorithms provides valuable insights for selecting the most appropriate ML approach based on specific industrial requirements and performance criteria.
