Self Driving Surveillance Robot Car

An intelligent autonomous robot car equipped with computer vision, real-time intruder detection, and notification systems that combines robotics, AI/ML, and surveillance technology for enhanced security monitoring.

Project Overview

This advanced surveillance robot demonstrates the integration of autonomous navigation, computer vision, facial recognition, and real-time notification systems. The robot car autonomously patrols designated areas while continuously monitoring for intruders using ML-powered detection algorithms and instantly alerts security personnel through mobile notifications.

Key Components & Features

Raspberry Pi 4 Controller and Arduino Uno

Serves as the main processing unit, handling computer vision algorithms, autonomous navigation logic, and real-time communication with notification systems that communicated in series.

HD Camera Module

High-resolution camera for real-time video capture, facial recognition, and intruder detection.

Ultrasonic Sensors Array

2 HC-SR04 ultrasonic sensors positioned strategically for obstacle detection, distance measurement, and autonomous navigation.

DC Motors with Encoders

Precision motor control system enabling accurate movement, turning, and positioning for autonomous patrol routes and obstacle avoidance.

WiFi Communication Module

Enables real-time data transmission, remote monitoring capabilities, and instant notification delivery to mobile devices and security systems.

Power Management System

Efficient battery management with voltage regulation ensuring extended operational time and reliable performance during surveillance missions.

AI & Computer Vision Implementation

Facial Recognition System: Utilizes OpenCV and haar cascade classifier for face detection to detect and identify faces in real-time, distinguishing between authorized personnel and potential intruders with accuracy.

Intruder Detection Algorithm: Each time a snapshot is been taken and the snapshot taken is verified against the databse

Real-time Notifications: Instant alert system that sends detailed notifications including timestamps and a message via Whatsapp to security personnel's mobile devices.

Autonomous Navigation: Reads sensor data that enable the robot to navigate by avoiding obstacles, and maintain optimal surveillance coverage of designated areas.

Technical Implementation

Control Architecture: Multi-threaded Python application running on Raspberry Pi 4, managing simultaneous processes for navigation, image processing, communication, and sensor data fusion.

Computer Vision Pipeline: Real-time picture processing using OpenCV for face detection, feature extraction, and comparison with authorized personnel database, achieving recognition accuracy of over 75%.

Communication System: Integrated messaging system using APIs to send instant notifications with captured images, detection confidence scores, and GPS coordinates to multiple recipients.

Skills & Technologies

Python Programming OpenCV Machine Learning Facial Recognition Autonomous Navigation Raspberry Pi Arduino Uno Real-time Processing IoT Communication Multithreading Sensor Fusion Mobile Integration
