Overview

Project Overview

This advanced surveillance robot demonstrates the integration of autonomous navigation, computer vision, facial recognition, and real-time notification systems. The robot car autonomously patrols designated areas while continuously monitoring for intruders using ML-powered detection algorithms and instantly alerts security personnel through mobile notifications.

When an unknown face is detected, the system immediately captures images, logs the event with timestamp and location data, and sends real-time notifications to security personnel via WhatsApp — enabling rapid response to security threats. The Raspberry Pi 4 and Arduino Uno communicate over serial for coordinated control.

AI Implementation

AI & Computer Vision Implementation

Facial Recognition System

Utilises OpenCV and Haar Cascade classifier for real-time face detection and identification, distinguishing between authorized personnel and potential intruders with over 75% accuracy under varying lighting conditions.

Intruder Detection Algorithm

Each time a snapshot is taken, the captured image is verified against the authorized personnel database. Deep learning-based face detection creates unique facial signatures for comparison with similarity thresholds tuned to balance security and usability.

Real-Time Notifications

Instant alert system sends detailed notifications including timestamps, captured images with confidence scores, and messages via WhatsApp to security personnel's mobile devices with encrypted transmission and multi-recipient support.

Autonomous Navigation

Multi-threaded Python application manages simultaneous processes for navigation, image processing, and communication. A finite state machine architecture combines high-level path planning with reactive obstacle avoidance using sensor fusion.
